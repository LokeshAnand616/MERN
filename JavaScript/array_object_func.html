JavaScript Advanced Concepts

1. Arrays

Explore advanced concepts of Arrays like `map()`, `filter()`, and `reduce()`.

const numbers = [1, 2, 3, 4, 5];

// Using map() to square each number
const squaredNumbers = numbers.map(num => num * num);

// Using filter() to get even numbers
const evenNumbers = numbers.filter(num => num % 2 === 0);

// Using reduce() to sum all numbers
const sum = numbers.reduce((acc, num) => acc + num, 0);

console.log('Squared Numbers:', squaredNumbers); // [1, 4, 9, 16, 25]
console.log('Even Numbers:', evenNumbers); // [2, 4]
console.log('Sum:', sum); // 15

2. Objects

Learn about advanced object handling like destructuring and object spread.

const person = {
  name: 'John Doe',
  age: 30,
  address: {
    street: '123 Main St',
    city: 'New York',
  }
};

// Object destructuring
const { name, age, address: { city } } = person;

// Object spread to create a new object
const updatedPerson = { ...person, age: 31 };

console.log('Destructured Name:', name); // John Doe
console.log('City:', city); // New York
console.log('Updated Person:', updatedPerson); // { name: 'John Doe', age: 31, address: { street: '123 Main St', city: 'New York' } }

3. Functions

Explore advanced function concepts like arrow functions and higher-order functions.

// Arrow function
const greet = name => `Hello, ${name}!`;

// Higher-order function: takes a function as an argument
const repeatGreeting = (greetingFunction, name) => {
  return `${greetingFunction(name)} ${greetingFunction(name)}`;
};

console.log(greet('Alice')); // Hello, Alice!
console.log(repeatGreeting(greet, 'Bob')); // Hello, Bob! Hello, Bob!

4. Promises and Async/Await

Learn about handling asynchronous operations using promises and async/await syntax.

function fetchData() {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      resolve('Data fetched successfully!');
    }, 2000);
  });
}

async function getData() {
  try {
    const result = await fetchData();
    console.log(result); // Data fetched successfully!
  } catch (error) {
    console.log('Error:', error);
  }
}

getData();

5. Classes and Inheritance

Learn about creating classes and inheritance in JavaScript.

class Animal {
  constructor(name) {
    this.name = name;
  }

  speak() {
    console.log(`${this.name} makes a sound`);
  }
}

class Dog extends Animal {
  speak() {
    console.log(`${this.name} barks`);
  }
}

const dog = new Dog('Rex');
dog.speak(); // Rex barks

6. Set and Map

Learn about the Set and Map data structures in JavaScript.

// Set example
const set = new Set([1, 2, 2, 3, 4]);
set.add(5);

console.log('Set:', set); // Set {1, 2, 3, 4, 5}

// Map example
const map = new Map();
map.set('name', 'Alice');
map.set('age', 25);

console.log('Map:', map); // Map {'name' => 'Alice', 'age' => 25}

7. Template Literals and Tagged Templates

Learn how to use template literals and tagged templates in JavaScript.

// Template literal
const name = 'Alice';
const greeting = `Hello, ${name}! How are you today?`;

console.log(greeting); // Hello, Alice! How are you today?

// Tagged template example
function uppercase(strings, ...values) {
  const result = strings.reduce((acc, str, index) => {
    return acc + str + (values[index] ? values[index].toUpperCase() : '');
  }, '');
  return result;
}

const message = uppercase`Hello, ${'Alice'}! How are you ${'today'}?`;
console.log(message); // Hello, ALICE! How are you TODAY?
